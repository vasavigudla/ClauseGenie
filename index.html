How to use Clausewise
  1. Upload documents
    Click the upload area to select files or drag and drop PDFs, DOCX, or images. After upload, you will see the files listed.
  2. Choose an output format
    Pick Clauses, Points, Divisions, or select Custom for specialized formatting.
  3. Custom output (optional)
    If you select Custom, choose a Style Template (required). You can optionally add instructions elsewhere in the app. Confirm to save.
  4. Run the analysis
    Click Start AI Analysis. Youâ€™ll see step-by-step processing with progress bars.
  5. View results
    The General Summary shows an overall narrative. If you selected Custom, a Custom Output tab will appear with results formatted to your chosen template (e.g., numbered, table, document summarization, named-entity summarization).
  6. Download or start new
    Use Download Results to save a report, or New Analysis to reset.
  7. Login and Library
    Login via the user icon to persist history. The Library shows past summaries: last 3 for guests, all for logged-in users.
Tips
  • Use sample documents to explore features quickly.
  • For Custom, choose the template that matches your goal (e.g., JSON or Table for structured exports).
  • If text appears faint, toggle theme; this UI is theme-aware.
